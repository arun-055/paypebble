💳 PayPebble – Retail Billing & POS System

PayPebble is a full-stack retail billing (Point of Sale) system designed to manage inventory, orders, and payments efficiently. It provides a seamless experience for managing products, customers, and transactions with a modern UI and secure backend.
🛠 Tech Stack

Backend:

Java
Spring Boot
Spring Security
JWT Authentication
Hibernate / JPA
MySQL

Frontend:

React
Redux
Tailwind CSS / Bootstrap
Axios

Payments:

Razorpay
